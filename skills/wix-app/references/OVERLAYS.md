# Overlay Compatibility Route

Choose by user context, not loose wording such as "drawer" or "panel."

| Need | Route |
| --- | --- |
| Selected desktop table record while retaining page context | [DASHBOARD_CUSTOM_TABLE_PANEL_PLAYBOOK.md](DASHBOARD_CUSTOM_TABLE_PANEL_PLAYBOOK.md) and WDS SidePanel documentation |
| Focused blocking task | [DASHBOARD_MODAL_PLAYBOOK.md](DASHBOARD_MODAL_PLAYBOOK.md) |
| Mobile sliding task surface | WDS Drawer documentation |

Do not use Drawer as a desktop SidePanel synonym or Modal as a selected-row fallback. The selected playbook owns composition and validation; installed WDS documentation owns exact props and examples.
