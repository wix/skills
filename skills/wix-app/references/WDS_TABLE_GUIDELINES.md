# WDS Table Compatibility Route

Use [DASHBOARD_CUSTOM_TABLE_PLAYBOOK.md](DASHBOARD_CUSTOM_TABLE_PLAYBOOK.md) for a custom WDS table, or [DASHBOARD_CUSTOM_TABLE_PANEL_PLAYBOOK.md](DASHBOARD_CUSTOM_TABLE_PANEL_PLAYBOOK.md) when selecting a row opens record detail. Those playbooks own table behavior and validation.

Do not apply either playbook to an Auto Patterns table. Auto Patterns behavior belongs to [DASHBOARD_AUTO_PATTERNS_PLAYBOOK.md](DASHBOARD_AUTO_PATTERNS_PLAYBOOK.md) and its installed references.
