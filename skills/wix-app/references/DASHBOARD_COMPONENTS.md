# Dashboard Component Documentation Map

Use this file only to find installed WDS documentation after a custom dashboard playbook has selected the behavior. Invoke the Wix Design System skill and inspect the exact source/examples before writing UI.

| Need | Documentation target |
| --- | --- |
| Custom data table | `Table`, `TableToolbar`, `TableActionCell`, Pagination, selected filter controls, and `EmptyState` |
| Selected desktop record | `SidePanel` plus Skin, Height, Header, Content, Footer, and Quick View examples |
| Mobile sliding work | `Drawer` |
| Form field | `FormField` and the selected input control |
| Feedback | `Notification`, `TopBanner`, or the selected documented feedback component |
| Page composition | `Page`, `Layout`, `Cell`, `Card`, and selected summary components |
| In-content separation | `Divider` |

For any unlisted need, search the installed `packages/wix-design-system` source and read the selected component's documentation and example. Record that target in the route plan. This map does not define behavior; the selected playbook does.
