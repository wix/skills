# Wix Design System — disabled for this variant

This app is generated **without** Wix Design System. There is no WDS component catalog,
no `wds.cjs` lookup, and no `@wix/design-system` dependency to rely on. Build every
dashboard/UI extension with plain React per the `wix-app` skill. Never import from
`@wix/design-system` and never add `@wix/design-system/styles.global.css`.
